# 个人应用SaaS平台
构建个人的应用平台，博客、微博、商城、论坛、直播、社交、IM等。      

从微博开始，一步步做成微服务，后续再做插件系统和应用市场。      

前端：vue、uniapp       
后端：hyperf、docker        


需要学习的知识点：      
* docker容器化，使用编排及部署
* 微服务架构，hyperf或者go语言
* 统一身份认证和授权
